# Python-App

![Progress](login.png)

## How to install Flask

```
cd Python-App
python3 -m venv .venv
. .venv/bin/activate
pip3 install Flask
```

## How to build the project

```
cd flask-tutorial
flask --app hello run
// for local network access
flask --app flaskr run --debug --host=0.0.0.0
```
